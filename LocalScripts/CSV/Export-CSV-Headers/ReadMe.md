
# URL-DownTime-Check
    
## SYNOPSIS

#Checks if the site is up with a 200 OK code.


## DESCRIPTION
        
Checks if the site is up with a 200 OK code. If not it exports a .csv with the date and the error.  
    
    
## Notes

Works in PS 5.1 and later.

## INPUTS
        
None.


## OUTPUTS
        
None.
