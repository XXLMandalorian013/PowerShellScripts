# Combine-CSVs-From-Dir

## DESCRIPTION

Grabs .csv's in a dir, sorts A-Z, weeds out repeates and combines them into one .csv.

## Notes

Works in PS 5.1 and later.

## INPUTS

None.

## OUTPUTS

None.
