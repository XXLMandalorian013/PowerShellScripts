# Role-Required-To-Rum-CMDLet

## DESCRIPTION

Gets all roles required to run a CMDLet in Exchange Online.

## Notes

ExchangeOnlineManagement Module only works in in PS 7.X.X.

## INPUTS

A CMDLet name.

## OUTPUTS

System.String.

## EXAMPLE

CMDlet: New-ComplianceSearch

Name           RoleType

----           --------

Mailbox Search MailboxSearch
