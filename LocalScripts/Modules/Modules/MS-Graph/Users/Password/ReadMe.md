# Update-MGUPW
## DESCRIPTION
Update an existing user's MS Graph Password.
## PARAMETER Name
Specifies the file name.
## INPUTS
None. You cannot pipe objects to Add-Extension.
## OUTPUTS
VERBOSE: Update-MGUPW has started
VERBOSE: Update-MGUPW has finished
