# WinGet Zoom Uninstaller

## DESCRIPTION

Uninstalls Zoom silently.

## Notes

Written for PS7.X so I dont have to install Nuget. Nuget is required for PS5.1.

## INPUTS

None.

## OUTPUTS

System.String :

Starting package uninstall...

Successfully uninstalled
